Solution

对于最小k个数这样的问题，通常有如下几种方法：

  1. 全部排序，提取前k个
  2. 对快排过程作改动，目标为锚点是第k个数
  3. 逐个比较添加至k个
  4. 维护大小为k的最小堆/优先队列

其中1､3的方法是比较直观的，2､4则是为了减少算法复杂度而分别对1､3作的改进。

本题在寻找最小k个数的过程中还需要先计算一个和，因此我们希望需要访问的数越少越好，而不是需要先计算所有组合的和。这样的情况下，3､4的动态方法明显比1､2好。而3､4的区别在于，一个是先比较再添加，一个是先添加再比较。下面分别说明用3､4的思想实现的题解。

Solution 1

逐个比较添加至k个

在这个方法中，我们需要考虑每次要比较什么样的数。

显然，若一个下标对已添加，那所有在它之前的下标对一定都已添加（因为它们对应的数都小于该数）。因此只需比较在那之后的下标对。

用一个数组记录num1中每一个下标对应num2中已添加的最大下标，然后下一次添加时往后找下标即可。

Solution 2

最小堆，先添加再比较

与方法1先找出最小的数再加入不同，方法2先把可能的数加入，再通过最小堆比较找出最小的数。
同样将两个数组排成上面所示的二维矩阵，发现有这样一个规律：前k个最小的数一定属于前k行(同时也属于前k列)，这是因为后面所有行的数均不小于前k行行首共k个数。换句话说，每次要增加一个最小数只需要增加一行（且取前k列）进入待选数（即入堆），并选出待选数里的最小数即可。

“ 0.1 “ => true

Note: It is intended for the problem statement to be ambiguous. You should gather all requirements up front before implementing one.

直接判断条件可以，但是太麻烦，各种if-else。

考虑把相同的代码块组合成函数，这样逻辑清晰。

一个有效的数具有如下结构：

其中[]表示可选，double指不包括e的浮点数。因此判断valid number的顺序就是首先去掉左右的空格（stripMargin），然后从最左边开始，先判断是否存在double（findDouble），如果后面接着’e’的话判断是否存在int（findInt）,最后到达最右边就是一个valid number。其中findXXX函数输入需要找的s子字段，如果没找到返回-1，否则返回找到后的下一个下标。

荒唐的羊

发表于 2016-01-13

看到网易云的记录，我才发现自己有多么脑残粉，去年听歌Top10里，居然有9首都是曾绵羊的。于是我努力回想着去年到底听了什么鬼——

然后又跑去循环了曾轶可的歌单。

绵羊有一首歌叫《荒唐的羊》，最开始听到时我是很想吐槽的，羊爱上狼这种老梗可以让我想到一百首神曲。然而后来这首歌我还是听了很多遍，只为了听那句：

爱上你 不因为你的美丽 不因为对的时机
不因为我有多孤寂
只因阳光很好 你走过我身旁
你说我荒不荒唐

听绵羊的歌也像是一件时不时想要做的荒唐事，虽然她非主流烂唱功简单英语，但就是没有防备的会把你感动到不行，真的是很特别的一个歌手。特别的地方还在于，她太难安利了，在很多人印象里就是一神曲歌手，既不流行又没有逼格。还好曾哥火的时候我还处于消息闭塞的高中时代，躲过了糟糕的印象，现在我的歌单里才会有这么一堆荒唐的趣味。

曾绵羊出的歌并不多，不算单曲和改版，迄今四张专辑，两年一张。09年的《Forever Road》是个无比傲娇的孩子，作为争议最大的时候出的第一张专辑，保留了很多青涩，没有这种趣味的人可能会听不习惯。当然也有人觉得还不够青涩，比如当时自称「罗永浩_可爱多」的老罗就不满高晓松的制作，自己做了更加恶趣味版的《还能孩子多久》和《勇敢一点》。而高晓松做的则是比较能让人接受的这10首歌：

  • 《最天使》
  • 《Forever Road》
  • 《电车计划》
  • 《白色秋天》
  • 《新的家》
  • 《Good Night》
  • 《还能孩子多久》
  • 《你是我最好的朋友》
  • 《视觉系》
  • 《狮子座》

这张专每首歌都很好，不知道从哪一首说起，所以就不说了。

11年的第二张专辑《一只猫的旅行》是21岁的曾轶可。那个傲娇的小孩长大了一点点，她说：

forever 21
我有很多事情要做
说对不起 说我爱你
都是我 都是我
——《Forever 21》

21岁的曾轶可，开始经常化妆：

有几次卸妆的时候 我受了点伤
脸上红红的地方 没事 一点小伤
你曾经吻我 也是这个地方
现在变成这样 心疼我的模样
——《卸妆》

21岁的曾轶可，开始学穿高跟鞋：

想要分一半的爱给孩子气
剩下的一半可不可以留给成熟的自己
穿上它踩着它
才会懂得 跌跌撞撞的意义
——《分一半的爱给高跟鞋》

21岁的曾轶可，开始疑惑那些挥霍的爱：

分分合合是我们
常做的不经大脑的事情
我们经常哭完笑
或者是笑着哭红了眼睛
——《每个路人熄灭一盏灯》

开始品尝爱的苦涩：

我需要现在 可你还不回来
我需要未来 可你都不敢猜
我需要大大的花园 里面种长长的誓言
有这么难吗 还爱不爱啊
——《我是你的，你是我的》

这张专辑里有几首命题作文。《Baby Sister》是快乐女声主题曲，《夜车》是王力宏导演的电影主题曲，《给你满分的我》是某段时间的麦当劳主题曲。最近还有一首命题作文《花样年华》是同名综艺节目的主题曲。这些歌其实都挺应景的，还不习惯她的同学可以从这几首开始食用，特别是《夜车》，吃过的人都说好。

13年第三张专辑《会飞的贼》高晓松又当回了制作人。这张专辑全是情歌，而且还是各种单恋的羊：

飞去太阳 偷走让人爱的阳光
飞去月亮 偷走让人怜的悲伤
飞去你心上 把阳光和悲伤一起释放
如果你还不懂 你还不要 你还不爱
那我就继续逃亡
——《会飞的贼》

在这张专辑里，曾轶可的词磨练得更好了：

oh 和你相爱就像一个秘密
无处藏身的秘密
我的心永远是一个秘密
它永远在想你
——《女人的秘密》

以及：

让蜡烛代替所有灯 让音乐代替话语声
此时无声胜有声
如果要我开口 只能说一句话
让我成为你的有可能
——《有可能的夜晚》

单恋的羊最后心累了，决定潇洒地离开：

我从你的身边开过也许你也不会记得
等雨后天放晴了 我就微笑着把你忘了
然后也不回头 再爱也不回头
这是我廉价的爱情开始珍贵的时刻
——《骑摩托的人》

还记得第一张专辑的摩托吗，那个不经世事的孩子才是真的潇洒：

想决定就决定了 没有下一步也没有为什么
想放弃就放弃了 放弃也是一种美丽呢
想说就说出口了 计划也算是誓言吗
电车计划 不惧怕雨水和太阳的谎话
——《电车计划》

经历痛苦的单恋后，按照两年一专的节奏到了15年的第四张专辑《25岁的晴和雨》。比较早的时候曾轶可有在尝试玩一些复杂的编曲，出了一首《风景》。那首我感觉太满了，像是刚开始玩，什么都想加上去。直到四专的《星星月亮》，就驾驭得很好了。作为新专第一首歌，《星星月亮》刚出来时是很大的惊喜，也预示着曾轶可在音乐上的转变和成长。专辑的发布方式很有意思，每月14号发布一张单曲，每张单曲的封面都是曾轶可自己画的画。

《25岁的晴和雨》这张专辑，「雨篇」是到了25岁终于开始感慨的曾轶可，她的词和曲变得工整，变成困惑的世人：

蜜蜂和花不定谁甜蜜了谁
男和女不定谁伤害了谁
我和你不定谁更爱谁
——《诗人的眼泪》

「晴篇」则更多的是尝试新的可能性，有对她一贯的小情绪在音乐上新的表达，有着重于编曲的《星星月亮》，还有《破车》这样讲梦想的歌：

坐我的破车 听一个笑话
说不定笑着 眼泪就来了
北京北京 无数个傻瓜
忘了流泪 也忘了回家
——《破车》

还记得第二张专辑的夜车吗，就是吃过都说好的那个。那辆车离开一个安静的小城，然后在大城里颠簸成了噼里啪啦的破车。就像潇洒的电车变成了决绝的摩托那样。

所以那个问着「我还能孩子多久」的小孩，终于还是长大了啊。

原来长大并没有那么可怕。

虽然有雨天的低落，但是可以在晴天看到很特别的彩虹啊。

虽然会经历想象不到的事，但还是可以长成自己想要的样子啊。

嗯，曾轶可我还可以再听100年。

2015观影回忆

发表于 2016-01-02

看了一下豆瓣记录，去年一年去影院看了11部电影，作为一个闵大荒的吃土少女，能保持一个月一部的频率已经是很难得了。主要归功于菁菁堂和碧江广场，极大地丰富了人民群众的文化生活。那么就来回忆一下这几部电影的观感。

【十二月】寻龙诀 ★★★★☆

导演：乌尔善
主演：陈坤/黄渤/舒淇/杨颖/夏雨
上映日期：2015-12-18

值得去电影院看，整部看下来很流畅，也有不少抓人的小点。

作为系列小说改编的电影，虽然是一个小故事但是背后有作者设定的大框架，所以不会有零散和不通的地方。故事构建基本按照冒险类型的模式来，几个性格各异有点小冲突或小暧昧的正派，用来推进或阻挠的大反派和小喽啰，过程惊险刺激，结局皆大欢喜。能把故事讲好，让观众得到快感，又不觉得被侮辱了智商，这部电影做到了商业片合格线上。

特效感人，很适合3D，时不时被吓到。还好它并不打算做成恐怖悬疑片，否则大晚上的墓地干尸，承受不来啊。

演技感人，几位主演都是让人可以期待的好演员。陈坤留了胡子完全认不出来，黄渤依然收放自如，舒淇依然很有风情。夏雨和他以前的形象完全不同，但是演得超级好，把小人物演得惟妙惟肖，一个大大的赞。

结尾差点以为陈坤真over了，心里想说不会吧主角诶，还好最后跳出来了。不过跳得有点强行HE，就是虐一下主角煽个情，然后天降主角光环。但是煽情那里促不及防被感动了，你知道我最喜欢你什么吗？最喜欢你骂我的样子，台词满分。

总之，期待做成像木乃伊的系列，期待下一部。

【十一月】我的少女时代 ★★★☆☆

导演：陈玉珊
主演：宋芸桦/王大陆/李玉玺
上映日期：2015-11-19

一个典型的开头，都市女白领疲于职场的烦心事，然后展开各种的回忆或者奇遇或者妖娥子。

类似于初恋这件小事，丑女孩为了心仪的男生变美，懵懂的爱情，错过的缘份，无数人的青春。不是所有女孩都是美的，但是所有女孩都有一个变美的过程。从大大咧咧的小屁孩到开始在乎身材在乎打扮在乎小眼睛黑皮肤，有一点小心思和一整套的内心戏，有时小心翼翼把一切藏着掖着偷偷失落，有时又不顾一切做一些特别夸张的事然后觉得自己傻到爆。青春就是这样一场拙劣的演习，当我们摆脱那种愚蠢时也就失去了这永远不能再现的冒险。

所以青春片特别容易引起大群人的共鸣，这并不能代表它多好，毕竟所有这种片子都有人痛哭流涕。当然在最近泛滥的青春片里，这部片还算处理得比较自然，属于诚意之作。

不过这也只能算是一剂麻药了。基于它的目标受众，男主基本是为了满足少女心而出现的，又帅又聪明，一个桀骜一个儒雅，而且还爱女主爱得深沉。特别是徐太宇，为女主默默付出拼命打架最后放手，像所有偶像片一样，少女心又一次得到了满足，却毁了一个原本可以是关于叛逆和成长的少年时代。当然导演本来也没打算有这一部分，只是各种脸谱化的角色让整个故事落入俗套，原本出彩的细节成了狗血剧情的补丁，我这种上了年纪的阿姨只好掩面看完。

看完的感觉就是，哦，我果然过了那个时代。不过还是有两个收获：之后那个周末跑去滑冰，get了新技能；以及再下一个周末用尤克里里弹唱小幸运，get了新曲子。

【十月】夏洛特烦恼 ★★★☆☆

导演：闫非/彭大魔
主演：沈腾/马丽/尹正/王智
上映日期：2015-09-30

是的这就是一部屌丝片。

一个完全的loser，每天浑浑噩噩靠老婆养活。有一天突然穿越到过去，靠着信息差咸鱼翻身，得名得利，迎娶梦中女神，走上人生颠峰。最后美梦破灭，男主回到糟糠之妻的怀中痛哭流涕。

马冬梅这样率性可爱的女人，比谁都懂得欣赏夏洛的才华和脾性，可惜夏洛一直没有真正懂得她的爱。对于这样的男人，当他怀才不遇的时候，家是供他求取和发泄的地方；意气风发的时候，家则是一个已经征服了的失去吸引力的地方。

这个故事告诉我们，不要傻傻为男人付出太多。就算已经付出了，也不要指望他可以回馈你什么。

【八月】滚蛋吧！肿瘤君 ★★☆☆☆

导演：韩延

第一次10km比赛

发表于 2015-12-13

先得瑟地晒一下我的奖状奖牌，以及感谢某人陪跑〜

昨天双十二，然而我除了全家以及晚上的火锅并没有凑任何热闹。小峰峰几周前怂恿我报这天的校园马拉松，一定是为了迂回地阻止我买买买，而我居然没有识破这一点，细思恐极(￣^￣)ゞ

话说回来，终于完成了一个10km比赛，实现了一个小小的愿望，还是很有成就感的。10km对我来说不是很难，至少在一个说得过去的时间内跑完没问题，不过一直都是自己在操场或跑步机上瞎跑。突然发现参加个比赛特别好，这样以后老了向子孙吹嘘时就可以拿出奖状奖牌，而不是寒寒碜碜地翻出一张咕咚的截图。何况这个奖牌那么重又那么好看，如果有个校园半程马拉松，为了奖牌爬着也要跑完(..•˘_˘•..)

其实我真正开始练习跑步应该算是去年，当时本科阶段基本尘埃落定，除了毕设没有什么其它压力。而我又是个闲不住的人，就觉得总得尝试点新的事情。当然主要还是因为我穷，不能到处游山玩水地浪起来，于是只好挥洒汗水，想象自己奔跑在宇宙中心。刚开始跑个10圈就觉得可厉害了，跑个7的速度就开始大喘气了，还伴随着各种膝盖疼脚踝疼趾甲疼。后来到处看帖子学习姿势水平，慢慢减少伤痛以及提速，入了门之后10km就只是时间问题了。这件事情告诉我们一个道理：贫穷也可以是第一生产力（咦？）。还有就是：

老赵的学生必须能跑(ง •̀_•́)ง

当然我现在的水平属于超超超业余，出于身体健康的目的没事这么跑跑应该够了。最近正在成形的目标是锻炼核心力量，主要是因为我穷，担心以后讨饭碗吃时没有搬砖的力气。核心力量的提高对跑步提速和运动损伤预防也有帮助。如果真能在老去之前再勾去两个小小的愿望：

  • 10km跑进60min
  • 完成一次半程马拉松

那对我来说真的是奇迹。但是谁知道呢，两年前10km也像是一个奇迹。

壮士，干了这碗鸡汤。

岁末

发表于 2015-12-10

迟钝如我，总是习惯以一副扑克脸面对生活的流转，然后在做出某种看似笃定的举动很久后才突然想到可信的理由，在脱离某些波澜不惊的时刻很久后才突然被隐蔽的情绪击中。就像现在处于回血模式的我——依然是一副扑克脸，呆坐于喧闹的生活中，想不出心中隐现的异样来自某月某日。

事实上，我甚至很难回想这一年的某月某日：它们比以往更快地离我而去，好似为了明证告别的坚决。最好我也同样坚决地离开这片废墟，抛却那种婆婆妈妈的缅怀和遗言，抛却乏味的所谓岁末总结，以及自以为的与过去握手言和的努力。可惜迟钝如我，只能更加茫然地回望，试图用沉默的过往给自己一些确信和勇气。

这一年至今，我认真记录自己心情的时候几乎屈指可数。不是因为改掉了多愁善感的坏脾气，只是到了这样一个尴尬的年龄，已经长大到那些小情绪开始显得无足轻重和矫情，又尚且稚嫩得说不出什么深刻的阅历和道理。何况迟钝如我，总是跟不上年龄增长的速度，总是一边拒绝着别人说的你应该，又一边怀疑着自己模糊的我想要。但是至少有一件事是我现在相信的：

我没有办法成为任何人，我只能成为我自己。
每个人独一无二的地方在于，他们必须完成自我的救赎。

这件事情让我想要开始记录，哪怕是琐碎的不如意的生活，和平庸的常常不知所措的自己。这应该可以算是一个新的开始吧，既然往事已经不可追，那么就以现在的确信印证确信，以现在的勇气印证勇气，以赤裸裸的自我，站在这个岁末告别和期冀。
