Leetcode - Valid Number

Validate if a given string is numeric.

Some examples:
“0” => true
“ 0.1 “ => true
“abc” => false
“1 a” => false
“2e10” => true

Note: It is intended for the problem statement to be ambiguous. You should gather all requirements up front before implementing one.

Solution

直接判断条件可以，但是太麻烦，各种if-else。

考虑把相同的代码块组合成函数，这样逻辑清晰。

Solution 1

一个有效的数具有如下结构：

valid number = [e]

其中[]表示可选，double指不包括e的浮点数。因此判断valid number的顺序就是首先去掉左右的空格（stripMargin），然后从最左边开始，先判断是否存在double（findDouble），如果后面接着’e’的话判断是否存在int（findInt）,最后到达最右边就是一个valid number。其中findXXX函数输入需要找的s子字段，如果没找到返回-1，否则返回找到后的下一个下标。

接着实现以上三个函数。其中对于findDouble和findInt，先判断第一个字符是否符号（[+,-]），如果是的话从下一个字符开始findUDouble和findUInt，否则从当前字符开始找。

对于findUDouble，如果首字符是’.’，则从下一个字符开始findUInt，否则从当前字符开始找[.[]]形式的数即findUDoubleD。对于findUInt，则是逐个查找是否数字字符即可。

Solution 2

Solution 1的缺点在于，要写太多函数层层深入。一种减少代码量和函数调用的方法是使用状态机，每读一个字符，根据状态机的转移矩阵判断当前读取数字进入哪一状态。其实Solution 1也是一个个状态判断，但是使用状态机可以把这些状态铺开，从而方便地用转移矩阵索引。
